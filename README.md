# Anima Della Moda

A professional multi-page clothing brand website created using HTML5 and CSS3.

## Project Overview

Anima Della Moda is a modern fashion brand website designed to showcase clothing collections in a clean and professional layout. The website focuses on minimal design, responsive layouts, and user-friendly navigation.

## Features

- Responsive design
- Multi-page website
- Semantic HTML5 structure
- External CSS styling
- Product showcase section
- Contact form
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub

## Pages Included

- Home
- About
- Products
- Contact

## Design Decisions

The website was designed using a minimal black and white color palette to create a clean and modern fashion aesthetic. Reusable CSS classes and responsive layouts were used to maintain consistency across all pages.

## Challenges Faced

One challenge during development was maintaining responsive layouts across different screen sizes. This was improved using media queries and responsive grid layouts.

## Deployment

The website was deployed using GitHub Pages.
